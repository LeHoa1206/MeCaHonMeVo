# Hướng Dẫn Thêm Hình Ảnh Cá Guppy

## Cách thêm hình ảnh của bạn:

1. Đặt các hình ảnh cá Guppy của bạn vào thư mục này (`public/images/`)
2. Đặt tên file theo format: `guppy1.jpg`, `guppy2.jpg`, `guppy3.jpg`, `guppy4.jpg`, `guppy5.jpg`, `guppy6.jpg`
3. Format: JPG hoặc PNG
4. Kích thước đề xuất: 800x1200 pixels (tỷ lệ 2:3)
5. Dung lượng: Dưới 500KB mỗi ảnh

## Tips để ảnh đẹp nhất:

- Chụp ở nơi có ánh sáng tốt
- Background đơn giản để làm nổi bật cá
- Focus rõ nét vào cá
- Chụp nhiều góc độ khác nhau
- Nén ảnh để tối ưu tốc độ load
